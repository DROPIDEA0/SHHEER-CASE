# SHHEER Case Website - Design Ideas

## Project Overview
A professional legal case documentation website for the SHHEER Bank Guarantee Dispute case. The site must present complex legal and financial information in a clear, chronological, and evidence-based manner.

## Idea 1: Legal Archive Aesthetic

**Design Movement:** Neo-Classical Legal Documentation with Modern Digital Archive

**Core Principles:**
- Authoritative and trustworthy visual language
- Clear hierarchical information architecture
- Evidence-first presentation with document authenticity feel
- Serious yet accessible professional tone

**Color Philosophy:**
- Primary: Deep Navy (#1a365d) - Authority, trust, legal gravitas
- Secondary: Warm Gold (#b7791f) - Importance, premium quality, official seals
- Accent: Crimson (#9b2c2c) - Critical evidence, warnings, key dates
- Background: Warm Ivory (#faf5eb) - Aged paper, archival quality
- Text: Deep Charcoal (#2d3748) - Readable, professional

**Layout Paradigm:**
- Vertical timeline as the central spine of the page
- Evidence cards that expand on interaction
- Sidebar navigation for quick section jumping
- Document-style sections with clear demarcation

**Signature Elements:**
- Embossed seal effects on key documents
- Timeline nodes with date stamps
- Evidence cards with "exhibit" numbering
- Subtle paper texture backgrounds

**Interaction Philosophy:**
- Reveal evidence on hover/click
- Smooth scroll-linked animations
- Download buttons with confirmation
- Lightbox for full document viewing

**Animation:**
- Fade-in reveals as user scrolls
- Subtle parallax on section backgrounds
- Evidence cards slide in from timeline
- Gentle pulse on critical evidence markers

**Typography System:**
- Headings: Playfair Display (serif, authoritative)
- Body: Source Sans Pro (clean, readable)
- Dates/Labels: Roboto Mono (technical precision)

## Selected Approach: Idea 3 - Olive Branch Justice Theme

This design best aligns with the Nesma Barzan brand identity (olive branches, mountain imagery) while maintaining professional legal documentation standards. The warm, natural color palette creates trust and accessibility, while the Mediterranean heritage elements add distinction and cultural relevance.

### Implementation Notes:
- Use the provided logo prominently in the header
- Create olive branch SVG dividers
- Implement a winding timeline path
- Design evidence cards as case file folders
- Maintain warm, inviting color scheme throughout
